# TPI-Programacion-II
DESCRIPCION
Food Store es un sistema de gestion desarrollado en Java mediante un enfoque basado en la Programacion Orientada a Objetos (POO). 

Este sistema nos permite administrar categorias, productos, usuarios y pedidos a traves de un menu interactivo por consola que a su vez nos redirigue a un menu CRUD (Create, read, update, delete) para cada uno. Toda la informacion se almacenara en memoria durante su ejecucion.

EJECUCION
Al ejecutar el programa lo primero que aparece es un menu de opciones:
1. Categorías
2. Productos
3. Usuarios
4. Pedidos
0. Salir

De las cuales podemos elegir escribiendo el numero correspondiente en la consola. Luego, siempre y cuando se ingrese una opcion valida, se desplegara un submenu que sera igual para cada una de las opciones anteriores (A excepcion de 0. Salir):
1. Listar
2. Crear
3. Editar
4. Eliminar

Este submenu nos permitira gestionar los objetos de las clases, agrupandolos en colecciones.
